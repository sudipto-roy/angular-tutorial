Angular 2 - Tutorials

Environment Setup

Required installations:

  • Install Node
  • install @angular/cli -g
  • npm install typings -g
  • npm install webpack -g
  • npm install webpack-dev-server -g

Steps to start a new Angular Project:

  • new name-of-project
  • cd name-of-project
  • npm install
  • ng serve

Angular Folder Structure

src
|- app
    |- common (optional and similar to core)
    |- core
    |  |- services
    |  |- constants
    |  |- core.module.ts (includes services and contants and exports them)
    |- root
    |  |- app-routing.module.ts
    |  |- app.component.html
    |  |- app.component.scss
    |  |- app.component.ts
    |  |- app.module.ts
    |- shared
    |  |- pipes
    |  |- directives
    |  |- components (that are shared throughout, like header/footer)
    |  |- shared.module.ts (includes pipes, directives and components and exports them)
